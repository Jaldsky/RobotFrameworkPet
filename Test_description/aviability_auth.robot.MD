# Test-suite ***aviability_auth.robot***
Тест-сьют содержит в себе два позитивных тест-кейса:
1. на проверку доступности vk.com 
2. на проверку авторизации по токену
<br>

Результаты выполнения тест-сьюта логируются в и доступны для просмотра по адресу: 
> /Results/logs/aviability_auth.robot

В папке ***aviability_auth.robot*** содержется три файла:
+ log.html - краткий лог
+ output.xml - лог в формате XML
+ report.html - подробный отчет

## Ручная проверка тест-сьюта
### Проверка доступности vk.com
1. Открыть браузер
2. Произвести GET-запрос, в адресную строку ввести: 
>  https://vk.com/

+ Ожидаемый результат: _код ответа страницы 2хх или 3xx

---
### Проверка авторизации по токену
1. Открыть браузер
2. Произвести GET-запрос, в адресную строку ввести: 
>  https://api.vk.com/method/account.getInfo?access_token=880e44a594166a1f7a1d10884139f111a8c9f39dc8027e5605d5c7114a05bc5d7240e42a39b505cb8eb2f&v=5.81

+ Ожидаемый результат: _появился json-файл, начинающийся с **response**_
